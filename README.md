# Scroll-Driven 3D Gallery Using a Blender Camera Path with Three.js and GSAP

Build an immersive 3D gallery where the camera travels along a Blender curve, with a dynamic focus effect on the images.

![Featured Image](https://tympanus.net/codrops/wp-content/uploads/2026/07/curvegallery_cover.png)

[Article on Codrops](https://tympanus.net/codrops/?p=117076)

[Demo](https://tympanus.net/Tutorials/CurveGallery/)

## Installation

```bash
npm install
npm run dev
```
